# 🚀 KONVERT CRM

### Built to Convert.

KONVERT CRM is a modern Sales Analytics Dashboard and Customer Relationship Management (CRM) platform designed for startups, sales teams, freelancers, agencies, and small businesses. Inspired by Bauhaus design principles and Gen-Z aesthetics, the application provides powerful lead management, sales tracking, business analytics, and Excel-based data handling without requiring a backend database.

---

## 📸 Project Overview

KONVERT CRM helps businesses manage leads, track sales activities, monitor revenue growth, and visualize performance through an interactive dashboard.

The application uses Excel files and browser local storage for data management, making it lightweight, portable, and easy to deploy.
